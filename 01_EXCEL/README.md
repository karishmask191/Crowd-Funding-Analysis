# 📊 Excel Dashboard – Crowdfunding Analysis

## 📌 Overview

This dashboard focuses on analyzing crowdfunding data using Excel.

It provides insights into project performance, funding patterns, and success trends.

## 🔍 Key Analysis

* Total projects and success rate
* Funding distribution across categories
* Year-wise project trends
* Backer contribution analysis

## 📈 Dashboard Features

* Interactive filters for category and time
* Clean KPI cards for quick insights
* Visual charts for trend analysis
* Easy-to-understand layout for business users
