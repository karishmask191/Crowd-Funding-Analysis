# 📊 Power BI Dashboard – Crowdfunding Analysis

## 📌 Overview

This dashboard provides an interactive analysis of crowdfunding projects using Power BI.

It focuses on identifying success patterns, funding trends, and key performance indicators.

## 🔍 Key Analysis

* Overall project performance and success rate
* Funding trends across categories and time
* Backer distribution and contribution
* Country-wise project analysis

## 📈 Dashboard Features

* Interactive filters and slicers
* KPI cards for quick insights
* Dynamic charts for trend analysis
* Drill-down capability for deeper exploration
