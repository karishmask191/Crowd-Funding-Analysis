# 📊 Tableau Dashboard – Crowdfunding Analysis

## 📌 Overview

This dashboard provides advanced visual analysis of crowdfunding data using Tableau.

It highlights trends, success factors, and category-level performance.

## 🔍 Key Analysis

* Success rate by category
* Funding distribution across goal ranges
* Year-wise project trends
* Country-wise project analysis

## 📈 Dashboard Features

* Interactive filters (year, category, country)
* KPI indicators for quick insights
* Visual storytelling with charts and maps
* Clean and intuitive layout
