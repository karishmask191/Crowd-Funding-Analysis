# 🗄️ SQL Analysis – Crowdfunding Data

## 📌 Overview

This module focuses on analyzing crowdfunding data using MySQL.

The objective is to extract meaningful insights using queries and data aggregation techniques.

## 🔍 Key Analysis

* Total projects and success rate
* Funding analysis by category and year
* Top successful projects by funding and backers
* Country-wise project distribution

## ⚙️ Techniques Used

* GROUP BY and aggregations
* Window functions
* Filtering and sorting
* Joins and subqueries

## 📁 Files

* SQL scripts containing all analysis queries

## 💡 Key Learnings

* Writing efficient SQL queries
* Handling large datasets
* Data transformation and analysis
* Converting raw data into insights
