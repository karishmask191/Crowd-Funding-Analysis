# 🚀 Crowdfunding Data Analysis (End-to-End Project)

## 📌 Project Overview

This project presents an end-to-end data analysis of crowdfunding projects using Excel, SQL, Power BI, and Tableau.

The goal is to analyze project performance, funding trends, and identify key success factors.

## 🛠️ Tools & Technologies

* Excel – Data Cleaning & Initial Analysis
* MySQL – Data Querying & Transformation
* Power BI – Interactive Dashboard
* Tableau – Advanced Visualization

## 📊 Key Insights

* Overall success rate
* Projects with lower funding goals show higher success rates
* Certain categories consistently perform better
* Strong geographical concentration of projects
